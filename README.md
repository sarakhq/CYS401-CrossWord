# 🔐 Cybersecurity Crossword Puzzle (CYS401 Project)

## 📌 Overview
This project is an interactive web-based **Cybersecurity Crossword Puzzle** developed for the **CYS401 course**. It is designed to help students reinforce key cybersecurity concepts through an engaging and user-friendly interface.

The application allows users to solve a crossword puzzle using cybersecurity-related terms such as *phishing*, *firewall*, *encryption*, and *malware*.

---

## 🎯 Objectives
- Reinforce cybersecurity terminology covered in CYS401  
- Provide an interactive and engaging learning experience  
- Transform theoretical concepts into practical activities  
- Enhance student retention through active participation  

---

## 🧩 Features
- Interactive crossword grid  
- Across and Down clues  
- User input validation  
- Real-time feedback (correct/incorrect answers)  
- Reset and show answers functionality  
- Clean and responsive user interface  

---

## 🛠️ Technologies Used
- **HTML** – structure of the application  
- **CSS** – styling and layout  
- **JavaScript** – logic, interactivity, and validation
